# kebbi pricing dashboard v11

## GitHub Pages 公開手順
1. この zip を展開
2. GitHub の公開用リポジトリ直下に `index.html` と `README.md` を配置
3. GitHub Pages を `main` ブランチ / `root` で有効化
4. 数分待って公開URLへアクセス
